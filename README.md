# Graph to Quiver Algebra

A single-page browser tool for drawing an undirected loopless graph and producing the associated quiver algebra data.

## Use

Open `index.html` in a browser, or publish this repository with GitHub Pages.

## GitHub Pages

In the GitHub repository:

1. Go to `Settings`.
2. Open `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Set `Branch` to `main` and `Folder` to `/ (root)`.
5. Save.

The page will be available at:

`https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/`
